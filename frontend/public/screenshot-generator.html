🛠️ How to Generate App Store Screenshots

Step 1: Click any "Capture" button above to open that screen in full iPad layout
Step 2: Use browser developer tools:
  • Press F12 or right-click → "Inspect Element"
  • Click the device toggle icon (📱) or press Ctrl+Shift+M
  • Select "iPad Pro" or create custom device with dimensions 2732x2048
Step 3: Take screenshots:
  • Chrome: Use "Capture screenshot" in dev tools menu
  • Safari: Use "Capture screenshot" in Web Inspector
  • Firefox: Use "Take a screenshot" in dev tools
  • Alternative: Use your system's screenshot tool after resizing browser
Step 4: Save images as PNG files with descriptive names:
  • 01-home-dashboard.png
  • 02-trainer-search.png
  • 03-health-tracking.png
  • 04-social-features.png
  • 05-progress-analytics.png
  • 06-session-checkin.png
