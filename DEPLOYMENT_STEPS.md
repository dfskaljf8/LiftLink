### **⚡ Quick Deployment Process:**

**Step 1: Login to Expo (On your local machine)**
```bash
npm install -g @expo/cli eas-cli
cd /path/to/LiftLinkMobile
eas login
# Use your credentials: ksurepalli259 / c.tqL2MD9B++?xv
```

**Step 2: Configure Project**
```bash
eas build:configure
# This will create your project in Expo and configure builds
```

**Step 3: Build for Both Platforms**
```bash
# Build iOS for App Store
eas build --platform ios --profile production

# Build Android for Google Play
eas build --platform android --profile production
```

**Step 4: Submit to App Stores**
```bash
# Submit to iOS App Store (requires Apple Developer account)
eas submit --platform ios

# Submit to Google Play Store (requires Google Play Developer account)
eas submit --platform android
```
